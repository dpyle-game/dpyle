How To Play

This game is inspired by Diffle.

Guess the word using as few total letters as possible.

Each guess must be a symbol exported from discord.py (a module name, class name, function name, method name, property name, etc). Hit the enter button to submit.

After each guess, the color of the tiles will change to show how close your guess was to the word.
